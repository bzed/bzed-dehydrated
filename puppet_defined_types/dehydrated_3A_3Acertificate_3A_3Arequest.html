Summary

Prepare everything to request a certifificate for our CSRs.

Overview

Does all the necessary things to be able to request a certificate from letsencrypt using the csr created and shipped to the dehydrated_host.

Examples:

    request_fqdn    => 'foo.bar.example.com',
    dn              => 'domain.bar.example.com',
    config          => {},
    dehydrated_host => host.that.runs.dehydrated,
}
